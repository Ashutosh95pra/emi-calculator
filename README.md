# 💰 EMI Calculator

A React.js-based EMI Calculator that helps users calculate:

* 📌 Monthly EMI
* 📌 Total Interest Payable
* 📌 Total Payment Amount

## 🚀 Features

* Responsive design
* Loan tenure support in Years or Months
* Instant EMI calculation
* Clean and user-friendly interface

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* Bootstrap
